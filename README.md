# M-S-Rebrand
This repository contains the initial GUI implementation for the M&S digital rebrand. The project’s primary objective is to modernize the M&S "legacy brand" identity to attract a younger "Under-35" demographic—aiming for a 15% growth in this segment within 12 months. The digital design prioritizes a "modern, tasteful vibe" while maintaining the quality trust established with the existing loyal customer base.

The repository is organized into a clean directory structure to facilitate scalability and high performance:

rebrand.html - Semantic HTML5 structural framework.

style.css - CSS3 styling, animations, and branding variables.

coursework.js - JavaScript for dynamic interactions.

Functional Features
Multi-Platform Visual Identity: Developed a unified system using SVG icons and a modern "M&S Green" color palette.

More Personalisation: Included asset planning for an updated "Sparks" loyalty interface that utilizes real-time trend data.

Performance (Scalability): Assets are optimized to maintain a sub-2-second load time on mobile devices, even during high-traffic periods.

Accessibility: All code adheres to WCAG 2.1 Level AA standards to ensure inclusivity for older demographics and users with disabilities.

Dynamic Engagement: Implemented a scroll-reveal system using a mathematical trigger point: Trigger = WindowHeight - 150px
